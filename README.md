## Ecommerce Website

### Features
- Product Display
- Product Search
- Login/Register
- Personal Cart for products

### Installation instructions 

##### Installing Node.js 

###### Follow the instructions to install node and npm.

https://nodejs.org/

##### Installing dependencies

```
npm install

```

### Runnning the server

```
node Website/bin/www

```

Open any browser and go to localhost:7999.
